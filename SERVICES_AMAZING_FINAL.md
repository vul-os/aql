- Text clips to gradient (bg-clip-text)

- Professional, modern, eye-catching

### 2. **Elegant Stats Bar** - 4 Cards

Each stat card features:
- **Clean white background**
- **Colored icon box** (emerald, blue, purple, slate)
- **Large bold number** (text-3xl)
- **Hover effect** - Shadow increases + trend icon fades in
- **Subtle border** - Not bright
- **Rounded corners** (rounded-2xl)
- **Professional spacing**

Stats Shown:
- 🟢 Active Services
- 🔵 Locations
- 🟣 Bots Deployed
- ⚪ Total Area

### 3. **Smart Search & Filters**

**Search Bar:**
- Icon inside (left)
- Large (h-12)
- Placeholder: "Search by service or location name..."
- Rounded-xl
- White background
- Subtle shadow

**Filter Buttons:**
- All / Active / Pending
- Shows counts
- Active state highlighted
- Rounded-xl
- Same height as search
- Clean pill style

### 4. **Beautiful Service Cards** - The Star!

**Card Structure:**
```
┌─ [Colored Line 1px] ──────────┐
│ [Icon●] Service Name      [→] │
│         📍 Location           │
│         [● Status]            │
├───────────────────────────────┤
│  🌱    🤖    📏              │
│  3     3     450             │
│  Gardens Bots Area           │
├───────────────────────────────┤
│  [Alert if pending]           │
└───────────────────────────────┘
```

**Visual Magic:**
- **Top border** - Service color (1px)
- **Hover glow** - Subtle colored overlay
- **Icon grows** - scale-110 on hover
- **Arrow appears** - Fades in on hover, slides right
- **Shadow lifts** - Elevates on hover
- **Status dot** - On icon (colored, pulsing)
- **Whole card clickable** - No buttons needed

**Color Coding:**
- 🟢 Lawn → Emerald
- 🔵 Pool → Blue
- 🟣 Security → Purple
- 🟠 Weather → Orange

**Each service type has:**
- Unique icon
- Unique color (subtle)
- Colored top border
- Colored icon background
- Consistent, professional
