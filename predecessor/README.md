# Predecessor artifacts

Salvaged fragments from the original BotKorp repo that lived only in deleted
branches (never in a branch tip), preserved here so nothing is lost now that the
source repo is gone. Not part of the current Aql app.

- `supabase-migrations/` — 14 superseded Supabase DB migrations from an alternate
  branch timeline. Secrets scrubbed. The full predecessor history is also on this
  repo's main chain and in the archived bundles.
